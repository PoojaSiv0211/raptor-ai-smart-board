Raptor AI Smart Board
Raptor AI Smart Board is an intelligent digital whiteboard built with Flutter and a Python backend. It allows users to draw on a board, circle handwritten content, and automatically generate concise study notes using AI and OCR.
The system is designed to support interactive learning by turning handwritten concepts into structured explanations and key points.

Features:

AI Circle Search
Users can draw or circle any handwritten word on the board. The system detects the text and generates short study notes automatically.

Handwriting Recognition
The backend uses Tesseract OCR to extract text from the circled region.

AI Study Notes
The detected concept is processed by the AI service to produce:

A short explanation

Key points useful for exams

Reference source

Smart Board Tools
The whiteboard includes tools for:

Pen drawing

Shapes

Erasing

Color selection

Stroke width selection

Insert to Board
Generated explanations can be inserted directly onto the board for further annotation.

PDF and File Handling
The system includes file storage and management for learning materials.
